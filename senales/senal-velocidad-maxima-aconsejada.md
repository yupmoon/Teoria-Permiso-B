# Señal de velocidad máxima aconsejada

Tags: #permiso-b #senales #velocidad #recomendacion

## Regla del esquema

La señal de velocidad máxima aconsejada se representa como una señal cuadrada azul con la velocidad indicada.

Indica una velocidad recomendada o aconsejada, no una prohibición como [[senal-velocidad-maxima]].

## En mis palabras

La clave visual es el cuadrado azul: no es la señal circular roja de velocidad máxima obligatoria. Sirve para aconsejar una velocidad adecuada.

## Idea clave para el examen

Velocidad máxima aconsejada = señal <ins>cuadrada azul</ins> = recomendación.

## Trampa habitual

Confundir "aconsejada" con límite obligatorio. Para límite obligatorio se usa [[senal-velocidad-maxima]].
