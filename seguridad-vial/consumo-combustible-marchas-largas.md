# Consumo de combustible y marchas largas

Tags: #permiso-b #seguridad-vial #consumo #combustible #marchas

## Regla

El consumo de combustible es menor si se circula con <ins>marchas largas</ins> y el motor <ins>poco revolucionado</ins>.

## En mis palabras

La idea del test es no confundir "marcha larga" con "motor muy revolucionado". Para reducir consumo, interesa una marcha larga, pero con el motor poco revolucionado.

## Idea clave para el examen

Menor consumo = <ins>marchas largas</ins> + <ins>motor poco revolucionado</ins>.

## Trampa habitual

Responder "marchas largas y motor muy revolucionado". La marcha larga ayuda, pero el motor muy revolucionado aumenta el consumo.
